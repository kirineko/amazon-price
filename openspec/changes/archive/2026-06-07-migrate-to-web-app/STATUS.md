# Status: 已放弃

**归档日期：** 2026-06-07

## 结论

服务器 Web 部署方案**已放弃并拆除**。

## 原因

- 部署于香港数据中心 IP，Amazon.co.jp 对自动化请求返回 **503 反爬拦截页**（非真实服务不可用）。
- 首页 WAF challenge、搜索页/批量抓取均不稳定，无法作为可靠生产方案。
- 即使修复配送地区初始化（商品页取 CSRF token），长期运行仍频繁触发 503。

## 处置

- 已删除：Docker、`docker-compose`、nginx 反代配置、`DEPLOY.md`、Axum Web 服务、单密码认证、SSE API、登录页。
- 已回退：**Tauri 桌面应用**为唯一交付形态。

## 参考

若未来需要远程访问，需考虑日本住宅/ISP 代理等方案，不在本 change 范围内。
