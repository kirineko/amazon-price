# Status: 已放弃（服务器部署上下文）

**归档日期：** 2026-06-07

## 结论

本 change 中针对**服务器海外 IP 部署**的价格抓取强化方案，随 Web 服务器方案一并放弃。

## 原因

- 核心改进（搜索页取价、JPY 锁定、币种校验、分批冷却）在**数据中心 IP** 下仍无法稳定通过 Amazon 503 拦截。
- 服务器端已拆除，不再维护远程部署路径。

## 代码处置

以下能力**保留在桌面版**抓取内核中（`region.rs` / `scraper.rs`），供本机 Tauri 应用使用：

- 商品页 fallback 取 CSRF token（规避首页 WAF）
- 搜索页优先 + 商品页 `psc=1` 兜底
- 币种校验、大批量分批冷却
- 复制价格保留小数

未完成的验收项（7.5 香港服务器部署、7.6 端到端）**不再适用**。
